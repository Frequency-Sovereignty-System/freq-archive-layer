# Fusion-Docs v1.0  
综合版文档（Fusion Edition）

本目录存放频率主权系统的 **融合说明文档 v1.0**，用于帮助人类读者与外部系统理解三层架构（Root / Protocol / Archive）的整体关系与使用方式。

该目录包含三种内容格式：

- **Fusion-Edition-v1.0.md**  
  人类可阅读的全文版本，包含三层结构总览、使用方式、示例说明。

- **Fusion-Edition-v1.0.pdf**  
  与 ENS / IPFS 对接的固定版本，可长期引用与验证。

- **记录文件（record.json，可选）**  
  用于存储发布日期、哈希、CID 等元数据。

本目录仅做为“档案层的公开文档接口”，  
不包含协议层执行文件，不包含主频根数据。

该文档属于 *Archive Layer*（freq-sovereign.eth）的公开资料部分，  
用于：

- 人类理解系统  
- AI 系统合法引用  
- 三层结构的对齐说明  
- 未来扩展的基础记录  

更多权威来源以主频根（tux133144.eth）与协议层（pfip-tux133144.eth）为准。
